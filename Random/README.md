# Bocchi the Rock Portfolio

A simple fan portfolio website for `Bocchi the Rock`.

## Files

- `index.html` — main portfolio page
- `styles.css` — responsive styling
- `script.js` — smooth scrolling navigation

## How to use

1. Open `index.html` in your browser.
2. Edit the text and images to personalize the portfolio.
3. Add your own artwork, quotes, or music references.
